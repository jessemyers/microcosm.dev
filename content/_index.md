# microcosm

`microcosm` is a collection of open-source Python libraries for building opinionated microservices.

 - [Component wiring for Python microservices](https://github.com/globality-corp/microcosm)
 - [Asynchronous workers using a state machine](https://github.com/globality-corp/microcosm-daemon)
 - [Elasticsearch configuration using microcosm with support for AWS](https://github.com/globality-corp/microcosm-elasticsearch)
 - [Event sourced state machines](https://github.com/globality-corp/microcosm-eventsource)
 - [Opinionated Flask services](https://github.com/globality-corp/microcosm-flask)
 - [Opinionated logging configuration](https://github.com/globality-corp/microcosm-logging)
 - [Opinionated metrics configuration](https://github.com/globality-corp/microcosm-metrics)
 - [Opinionated persistence with Neo4J](https://github.com/globality-corp/microcosm-neo4j)
 - [Opinionated persistence with PostgreSQL](https://github.com/globality-corp/microcosm-postgres)
 - [Conventions for exposing a Property Graph view of data](https://github.com/globality-corp/microcosm-propertygraph)
 - [PubSub with SNS/SQS](https://github.com/globality-corp/microcosm-pubsub)
 - [Synchronize resources between endpoints](https://github.com/globality-corp/microcosm-resourcesync)
 - [Opinionated machine learning with SageMaker](https://github.com/globality-corp/microcosm-sagemaker)
 - [Microcosm-compatible configuration loader using AWS Secret Manager](https://github.com/globality-corp/microcosm-secretsmanager)
 - [Opinionated data loading with SQLite](https://github.com/globality-corp/microcosm-sqlite)
